# Food Chain Song
Generate the lyrics of the song 'I Know an Old Lady Who Swallowed a Fly'. Doing that aims to help us to learn more from Sandi Metz book "99 Bottles of OOP"

How to run the test
# Install minitest
gem install minitest

# Run the tests
ruby food_chain_test.rb
